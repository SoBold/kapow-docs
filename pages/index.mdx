# Kapow Docs

Welcome to the documentation site for the Kapow project! This documentation serves as a guide for developers on Kapow.

## Contributing

To contribute to this docs site follow these steps:

- Pull the repo [Kapow-Docs](https://github.com/SoBold/kapow-docs)
- Make changes to `./pages`
- Commit following [Conventional commits](https://kapeli.com/cheat_sheets/Conventional_Commits.docset/Contents/Resources/Documents/index)
- Push to the repo
